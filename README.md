# Мониторинг и наблюдаемость ML-системы: Virtual Product Placement

Этот проект посвящен реализации системы мониторинга и наблюдаемости для ML-сервиса, работающего по архитектуре Kappa для встраивания рекламы в видеопоток.

## Структура репозитория
- `prometheus.yml` — конфигурация сбора метрик.
- `dashboard.json` — экспортированный дашборд Grafana.
- `drift_report.html` — отчет об обнаружении дрифта данных.
- `HW8_monitoring.ipynb` — основной ноутбук с реализацией.
- скриншоты работы системы и алертов.

## 1. Определение SLO (Service Level Objectives)
Для нашего ML-сервиса установлены следующие целевые показатели:
- **Latency (p95)**: < 1 секунды на инференс кадра.
- **Error Rate**: < 1% от всех запросов.
- **Availability**: > 99% времени работы системы.

## 2. Мониторинг (Prometheus & Grafana)
Система мониторинга развернута с использованием Prometheus для сбора метрик экспортера (порт 8000) и Grafana для визуализации.
- **Метрики**: Используется `Histogram` для отслеживания квантилей задержки.
- **Дашборд**: Включает графики p95 latency и интенсивности запросов (RPS).

## 3. Обнаружение дрифта (Evidently AI)
Реализован механизм сравнения эталонного (Reference) и текущего (Current) пакетов данных. 
- Результат: выявлен дрифт в признаках задержки и пропускной способности, что сигнализирует о необходимости переобучения модели или проверки инфраструктуры.

## 4. Архитектура системы (Kappa VPP)
Система обработки видео реализована на базе архитектуры Kappa:
- **Ingestion**: Kafka/Redis Broker принимает видеопоток.
- **Processing**: YOLOv11 выполняет детекцию объектов (людей) в реальном времени.
- **In-painting**: Модуль генеративного ИИ накладывает логотипы на одежду.
- **Output**: Обработанный поток сохраняется или передается далее.

## 5. Инструкция по запуску
1. Установите зависимости: `pip install diagrams evidently ultralytics faststream[redis]`.
2. Запустите Redis и Prometheus серверы.
3. Выполните код экспортера метрик в ноутбуке.
4. Импортируйте `dashboard.json` в Grafana для визуализации состояния системы.
5. Запустите VPP-конвейер для обработки видео `HW8_building_and_person.mp4`.
